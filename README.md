# WoxZeal

need to install python, zeal (https://zealdocs.org/) and some docsets for zeal.

## config

### docspath

The path to zeal's dossets path:

```json
{
    "path": "D:\\Program Files\\Zeal\\dossets"
}
```

## help

```
z ~help
```

### set query alias

```
z ~set py3=python3
```

### del query alias

```
z ~del py3
```

### list query alias

```
z ~list
```
